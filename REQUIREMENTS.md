# 投资页面功能清单

## 数据区（服务端预载，立即显示）
- [ ] 投资总览卡片（总市值/总成本/总盈亏/当日涨跌/持仓标的数/累计手续费）
- [ ] 投资分析（饼图：资产类型占比 / 折线图：收益率vs QQQ/SPY）
- [ ] 持仓组合表格（代码/名称/类型/数量/成本/现价/涨跌%/盈亏）
- [ ] 投资账户卡片（名称/可用现金/持仓市值/转账按钮）
- [ ] 交易记录表格（日期/类型/标的/数量/单价/金额/平台/编辑删除）
- [ ] 定投计划卡片（名称/类型/频率/金额/费率/开始日期/已投期数/投入/份额/均价/市值/收益率/按钮）

## 交互功能（JS动态）
- [ ] 添加交易表单（日期/类型/资产类型/标的/数量/单价/手续费/总金额/币种/平台/账户）
- [ ] 编辑/删除交易记录
- [ ] 刷新行情按钮（调外部API拉最新价）
- [ ] 定投添加/编辑弹窗（标的/类型/方式/金额/数量/费率/币种/频率/节假日/开始/结束/无限期/下次执行/平台/投资账户/扣款账户）
- [ ] 定投暂停/恢复/删除
- [ ] 定投立即执行（手动触发一次）
- [ ] 历史补投（开始日期在过去→自动生成买入记录）
- [ ] 账户转账弹窗（从/到/金额/币种/日期）
- [ ] 标的自动填名（内置映射+Yahoo Finance）
- [ ] 代码类型识别（数字=CNY/基金, 字母=USD/股票）
- [ ] 自动算数量/金额（按金额定投时算qty, 按数量时算金额）
- [ ] 60秒自动刷新（仅读缓存，不调外部API）

## 安全
- [ ] SQLAlchemy参数化查询防注入
- [ ] Pydantic校验所有写入
- [ ] Jinja2 autoescape防XSS
- [ ] 预载数据为服务端生成JSON，非用户输入

## 性能
- [ ] 首屏数据服务端预载嵌入HTML，0个API请求
- [ ] 只有点"刷新行情"才调外部API
- [ ] 60秒定时刷新只读本地缓存
